2018-09-11記載
媒質：ランダム配置の円形EMCL
素子数：100 x 2
タイプ：二平板
TOFマップ作成法(波形先頭抽出法)：半自動AIC法
問題：EMCLが介在せず，明らかに経路に変化が生じないであろうところにおいても経路変化があった．しかも経路はクロスしている．この原因を探る．
対策：TOFマップとRFデータを見る
結果：AICの評価区間が十分広くなかった．あるいはP波の有無によって大きく影響を受けていると考えられる．
考察：AICは統計的手法であり，物理特性を考慮した波形先頭抽出方法とは言えない．そこで，しきい値法を臨時で用いる．
　　　問題点としては，実験をする際にしきい値法ではきっと誤差が大きくなるだろうということである．
　　　位相に着目した先頭抽出法が求められる．
